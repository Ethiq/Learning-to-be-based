# Day 1 — What is Base?

## The Basics

Base is a **Layer-2 blockchain** built on Ethereum by Coinbase.

A Layer-2 means it:

- Runs on top of Ethereum
- Makes transactions faster and cheaper
- Still inherits Ethereum's security model

Think of it like:

> **Ethereum** = the main highway
>
> **Base** = a fast express lane built on top of that highway

Transactions on Base get bundled up and settled back to Ethereum L1, so you get the speed benefits without giving up the security guarantees of the base layer.

## Why This Matters

Most people can't afford to use Ethereum directly — gas fees can spike to $50+ for a simple swap. L2s like Base bring that cost down to fractions of a cent while keeping the same trust model.

## Key Takeaway

Base isn't a separate blockchain competing with Ethereum. It's an extension of Ethereum that makes it actually usable for everyday transactions.
